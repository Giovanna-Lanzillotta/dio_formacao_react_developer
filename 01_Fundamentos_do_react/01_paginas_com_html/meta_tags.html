content-language

Atenção: essa tag está caindo em desuso! Atualmente, a W3C (quem manda nas regras da web) recomenda que você defina o idioma diretamente na tag html, lá no topo do seu arquivo. É muito mais eficiente para o Google e para leitores de tela.


http-equiv="content-type" content="text/html; charset=iso-8859-1"

É uma forma antiga de definir a codificação de caracteres do seu site. No entanto, ela usa o padrão iso-8859-1 (também conhecido como Latin-1), que é bastante limitado.


Author

Ela serve para declarar oficialmente quem foi a pessoa responsável por desenvolver aquela página ou projeto.


http-equiv="refresh"

Essa tag é o "temporizador" do HTML. Ela serve para fazer duas coisas automaticamente: recarregar a página ou redirecionar o usuário para outro site após alguns segundos.
