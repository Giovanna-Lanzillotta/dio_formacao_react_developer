Seletores

Em CSS, um seletor é a parte do código que "seleciona" qual elemento HTML você deseja estilizar. Ele funciona como um endereço: você diz para o navegador "pegue todos os elementos X e aplique estas cores e tamanhos neles".

Exemplos de seletores

Tipo
Afeta todas as tags daquele tipo na página.
Classe
É o mais usado no dia a dia (e no React). Você cria um nome e aplica a qualquer tag no HTML usando class="nome". No CSS, ele sempre começa com um ponto.
ID
Seleciona um elemento único na página. Não deve ser repetido. No CSS, começa com uma hashtag.
